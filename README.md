Unismart Store – QA SDLC Report

Project Overview This project focuses on testing a web-based e-commerce application where users can browse products, search by categories, add items to a cart, and complete purchases online. The system includes features like user registration, login, product search, product details, cart management, checkout, payment, and user dashboard.

SDLC Phases 1 Requirement Analysis 2 Test Planning 3 Test Design 4 Development 5 Testing 6 Deployment

Functional Modules 1 User Registration – User signup with basic details 2 Login & Logout – Authentication functionality 3 Product Search – Search products 4 Product Details – View product information 5 Cart Module – Add and manage cart items 6 Checkout Module – Place orders 7 Payment Module – Complete payment 8 Order Confirmation – Confirmation after order placement 9 User Dashboard / Orders – View past orders

Non-Functional Requirements 1 Performance: Pages should load within 2–3 seconds 2 Usability: Simple UI and easy navigation 3 Reliability: No crashes and proper data handling 4 Security: Protect user and payment data

The system has been tested across all modules including registration, product browsing, cart management, checkout, and payment. Most functionalities are working as expected. The system is stable and ready for deployment.
